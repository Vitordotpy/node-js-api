# Api simples em NodeJS

## Pré Requisitos
**1 - Node**

## Como Executar
**1 - execute o terminal no diretório do arquivo index.js**  
**2 - execute o comando**
```
node index.js
```
## Métodos
**1 - GET - http://localhost:3000/users/list**  
**2 - POST - http://localhost:3000/users/create**  
**2.1 - params**
```
{
  cpf:'string',
  nome:'string',
  data_nascimento:'string'
}
```
